## 16级保研经验分享 南京大学计算机学院 学硕 Tonggege

感谢Zilize学弟的邀请，借此机会写一下2016级保研经验以及学院的保研情况。（以下是我的看法，如有不同观点欢迎讨论）

#### 计算机学院整体保研情况

2016级计院，在加权排名前50中，有15人左右出国，3-5人工作，10人保研外校，其余20人左右保研本校。

2016级计院，在加权排名前100中，有30人左右出国，10-20人左右工作，50人左右保研本校，10人保研外校。

以上出国数据根据放弃提交CSP成绩的同学人数推测，工作的人数是打听来的，保研外校是和我一起参加夏令营以及夏令营官网的offer名单中看到的，保研本校是所有人上面的人数算出来的。

本校保研主要的两个去向：1. 国家光电实验室 2. 东五楼金海老师实验室 。据我了解这两个实验室最抢手，在保研CSP上机成绩还没出的时候就已经有很多人抢先预订了，等出成绩后很多坑已经被占满。所以想保研本校的同学请在得知自己有可能保研的时候就要去联系实验室老师了。到目前为止我了解到的情况是，国光那边可以提前选导师，只要你有保研资格，和老师内定好，你的导师到时候就不会换。而东五楼不能够最后确定导师，你提前联系导师，导师会尽可能得申请你成为他的学生，但是据说某些情况还是存在最后得导师不是自己联系的。如果你打算去国光或者东五，我建议在暑假实习的时候就不要找校外企业去实习了，最好选择在实验室实习，提前和导师进行联系。

相对本校保研，外校保研了解的更多。首先任何一所学校的外校保研都是在所有保研里面的人数最少的，甚至比出国的人还要少(得多)。华科计院今年保外一共10个人（包括计科+ACM+校交+物联网）。（当然，东南大学、北邮、成电他们保外情况也和华科差不多）清华深圳研究院硕士两人，北大本部直博1人，人民大学信息学院1人，南京大学计算机学院1人，中国科学院计算技术研究中心4人，中国科学院信息工程研究院1人。今年没有去上交和浙大的，往年一共可能有两三个去上交浙大的。（我本人保研到南京大学计算机学院）

对于保外，我觉得影响因素排名有：1. 成绩排名和是否有论文 2. 是否有直博意愿(影响巨大) 3. 上机能力 4. 知名竞赛得奖与英语能力 5. 实习经历 (实验室或企业均可，能开出证明来是最好的) 6. 专业课的软硬实力（就是对计算机整个体系的理解及知识面的拓展）。

另外，保外最难的一关是夏令营入营申请，这一关能够刷掉90%的人，其中几乎主要看上面影响因素的1(90%)和4（10%）。等参加夏令营时，上面的2，3，5，6就非常重要，1和4基本上就不怎么看了。按照今年的情况，我了解到很多人都报名申请夏令营，但是能入营的人很少，基本上排名30以后的人很难入营。其中有一个排名45+的一个同学去了清华深圳研究院是因为他选择的是设计类方向的研究生（和我们专业相关性不大），和腾讯公司联合培养，招生方式和常规的夏令营方式不同。其余同学均排名在30名以内(计科+ACM+校交)。

#### 我的情况

加权：88.x（忘了……），加权排名：27/342，保研综合排名：第6（总人数未知），四级：542，六级：497，美赛H奖、华中赛二等奖（大一时拿的），字节条跳动实习三个月，各个实验室实习若干，CSP：280，国家级大创项目1个，做数据结构助教若干次。

- 夏令营申请：上交电院硕士，复旦NLP方向硕士，南大计算机硕士，中科大计算机硕士，人民大学信息学院硕士，浙大计算机硕士，中科院软件所硕士。

- 夏令营入营：南大计算机硕士，中科大计算机硕士，人大信息学院，中科院软件所
- offer：南大计算机硕士，中科大计算机硕士

其中人大信息学院我在waiting list，但是每年最后鸽人大的人不少，一般都能排到，我当时已经拿到中科大offer了，所以直接就发邮件让他们把我去掉。中科院软件所是因为他们建议夏令营期间私自联系老师，但是我就只报个名参加考核，所以没通过。我上机成绩是考核中的第9，理论成绩是考核中的第6，一共20+个人参与考核，预录取8人左右（但实际上最后只录取了6个）。

上交电院申请是硕士和博士二选一的，选了硕士就不能选直博反之亦然。ACM班的一些同学都选择了直博，我当时想着冲一下选择了电院硕士，果不其然被刷。我身边选直博的都被刷了更何况硕士。个人觉得上交是除了清华以外最难进的一个，硕士往年也几乎没有，我听到的去的都是直博。我一同学排名40左右报了上交软院硕士过了，所以到时候想去上交的人，选择报名的时候要慎重。

复旦我只想去NLP实验室，其他的方向个人觉得还不如华科呢。然后复旦NLP方向巨火热，而且还喜欢招数学系的学生，导致我没听说周围谁去了复旦NLP，我的一个群里面也没几个人去。倒是图像等方向有入营的。复旦招的人也很少，导致莫名其妙成了竞争相对比较激烈的一个。

人大计算机是我参加的第一站，大概就是第一天报到，第二天上午考察英语口语，下午考察上机，第二天上午考察专业课面试，然后就结束。英语口语首先是自我介绍，然后面试官用英文问问题，我再用英文回答。这个英语口语是我参加面试以来最差的，自我介绍中途卡壳（因为没提前准备，大家不要学我！！！），然后面试官示意我坐下，后面问我喜欢哪种运动，我没听清楚，面试官重复，我又没听清，又重复，最后换个问题问我喜欢什么样的音乐（对，就是这种初中问题），我一激动忘了民谣怎么说，内心毫无波动，甚至想夺门而出。最后用中文回答民谣（内心毫无波动，甚至想找个 地缝钻进去），面试官说folk music（我可能三年内都不会忘了这个词的）。经历了极其糟糕的英语面试，我内心毫无波动，甚至想去实习，去见见我PCG部门的leader，他微信已经联系我好几次了。下午上机分为两个，一个电脑上机，一个手写代码。电脑上机第一题是斐波那契数列，5分钟秒掉，第二题320位的十进制转换成二进制，也就是大整数的除法算余数和商。这个因为之前复习时关注点都在动态规划，搜索和图论了，这边就没怎么复习。所以写了20几分钟，最后一刻交上去，AC了。笔试的就是用纯C语言（不能用STL）实现链表、二叉树等等算法，三道题，因为我数据结构学的不错，所以这些不是难题。第二天面试，因为之前有今日头条3轮面试，腾讯4轮面试，对于专业课基础面试和项目经历面试来讲，我觉得都不难。平时课下我也会看一些更深一点的资料，知识面应该是足够应付课程以及课程的延拓了。专业课面试首先问了项目经历，讲解一下项目，然后问了几个专业问题（忘了……）就结束了，大概10分钟左右。然后就收拾东西去下一站了。人大参加完后我不是很确定自己能不能录取，上机和专业课不错，但是英语面试实在是太差了，比初中生还差那种，所以就有点悬。人大我也联系过老师，我所联系的老师也有考核，只有手写代码，一道简单的搜索，用自顶向下递归解决的，还有一道是数值分析，用编程求一个函数的导数，并保证误差在0.0005以内，速度越快越好。（数值分析在夏令营里面的考核也相当重要）。

中科院软件研究所是夏令营第二站。我本来没有打算报中科院软件所的研究生，一次偶然的机会，我参加了他们来华科的一个招生宣讲。招生宣讲上只有10来人，其中计算机学院的只有我一个，当时这个信息都在年纪群里发了，我也奇怪为啥只有我一个人来，其他的都是数学学院、软件学院、电信学院的人。因为招生宣讲来的人少，软件所的老师就答应了现场来的所有人一定保证入营。老实讲，当天零零星星的几个人着实看起来冷冷清清，我在台下莫名得感觉到有点心酸？后来果然我入营了，那些没去参加招生宣讲的人很多都没过……（也可能大佬没报）。当时想着既然入营，而且包车费（重点），就去了。软件所没有统一得考核，一律按照实验室招生。第二天的夏令营大宣讲会上公布各个实验室的招生人数，然后学生可以在随后的两天自行联系实验室进行考核，实验室同意录取并且获得学校的保研资格，那么你就有学上。我因为不打算去中科院读书，所以就随便的参加了一个实验室考核，最后没联系老师就没过。软件所相对于计算所要次一点，但是入营的学生也基本上是985或者强一点的211，其中也有清华和北大的学生参加。印象比较深的是北邮、成电和南开的人相对多一点，华科计院只有2个人入营。我参加的那个实验室是人机交互实验室，分为上机、笔试、面试，上机就是2. 5小时做4个题，一道是送分题，一道是普通搜索，一道动态规划，一道暴力搜索。

第三站是南京大学计算机学院。南京大学是我最想去的学校，所以在去的路上我还是有点激动的。从北京到南京的动车上，我一直在思考人生，想着当时前途迷茫，未来三年何处去。到了南京大学仙林校区，觉得这个学校很现代化，和南一楼形成了鲜明的对比(emmmm)，当时拖着行李走到南京大学计算机系楼，莫名的觉得自己有点稳。报道的时候是实验室介绍，给出了很多展板，每个展板介绍一个实验室，旁边还有相应的微信群二维码。不得不说南京大学实验室人工智能、自然语言处理、数据挖掘、程序形式化等等算法方面的几乎占了80%，只有一两个实验室和云计算等有关系。因为早晨报道，下午1.30就要上机，时间特别紧张，所以很多人包括我连行李都没放，直接放在办公室里。很快吃完饭就要去上级考核了，听说南大上机考核占60%-80%，反正就是很重要。因为有前两次的上机经验以及之前准备过两周的上机，所以我还是有一些信心的。一共有三道题，一道是小学语文老师都没描述清楚的送分题（后来统计这道题得分率最低），这道题当时题目没有描述清楚，导致我得了60分后始终找不到为啥40分没有，后来发现是他没有描述清楚，考完后其他人这道题也没怎么AC，和我差不多的分。第二题是动态规划，但是因为我在第一题耗时太久，所以就直接暴力＋剪枝解决的，最后只有７０分。第三题是二叉树给出前序和后序，求中序有多少种情况的，因为数据量只有３０所以我直接暴力搜索，这道题是满分，最后一共230分.南大上机可以实时看到自己多少分,题目是一个用英文描述的pdf文件. 考完后我觉得自己有点不稳,第一题那么简单肯定一堆人AC，心里有一丝丝难过．然后就下大暴雨了，打雨伞都没用的那种大暴雨，安排的酒店是四星级酒店，虽然在南大里，但是离计算机系楼太远了（南大很大，感觉和华科差不多），于是就在大暴雨中一个人前行．到了酒店，和西电的一个室友一起，我问他多少分，他说１７０.然后他说一同来的也没几个过２００.当时想着２３０看来还是个相对可以的成绩了．第二天是面试，南大的面试问了很多专业基础的东西，比如NP与P类问题是什么意思，如何加快数据库查询的速度（我回答的是数据分块, 数据先Ｌｏａｄ到缓冲区, 查询优化, 索引）,然后考官问了我很多索引的问题, 比如树索引和哈希索引,还问了树索引一些相关的问题. 最后就是用英语介绍一下你所熟悉的编程语言.我介绍的是C++,大概就是说C++可以操作计算机底层,是一种编译型(compiling)语言,它实现多态(multi-state)是通过虚表(virtual table)等等. 面试完后,我觉得差不多稳了,开开心心的在南京逛了逛.然后就去参加中科大的。

中科大站。因为南大我觉得有九成把握可以拿到offer,所以中科大这边可能就是走个过场. 中科大老师说东南大学往年鸽的人太多,所以前三年对东南大学的学生封锁了,从今年才第一批放东南大学进来.所以夏令营不参加的请明确说不参加,否则很可能会被封锁,影响下一届.，这是中科大招生老师亲口说的。中科大分为两部分，首先是学院的上机+面试考核，如果通过且有保研资格那么你肯定有中科大计算机学院的学上，然后是好的实验室单独考核，如果学院考核过了，且实验室考核过，那么你可以在该实验室读书。学校考核首先是上机，上机一共5道题，3.5小时。第一题300位m进制的数转换成n进制，比人大的难一些，其中需要一些trick。这个题我当时思路清晰，20分钟解决。第二题暴力搜索，20*20棋盘，一个子只能走(日)字，其他有些什么墙壁什么的条件不能走，问从a点到b点最少走几步。这个题也很快20分钟解决。第三题动态规划，我才用自顶向下设计，差不多也20分钟解决。第四题也是个搜索问题，具体忘了。第五题图着色问题，因为之前没有复习到图着色问题，所以就直接暴力解决，假设一个点是什么颜色，其余...，看成立不成立。这个题30分钟写完。最后提前了一个半小时写完。但是我看其他人还在写，就在纳闷什么情况，是不是有什么坑我没看出来，但是又把题看了一遍，检查了，构造很多样例都没问题，然后我就发呆。结束后有人问我那个进制转换怎么做，我就说了大整数除法什么的，注意一些trick就好了。第二天面试，果然华科和中科大都比较喜欢注重计算机理论知识，考的都是什么霍夫曼树、操作系统什么的，恰好我数据结构和操作系统是学的做好的两门，所以面试基本上没任何压力。当然中科大面试还会考核项目经历、实习经历什么的，这些如实说就好了。实验室，我报了一个机器学习的，还算可以的实验室，在中科大参加的第三天，南大给我发预录取的通知了。参加完中科大夏令营，就马不停蹄的去深圳实习了，在火车上中科大也给我发预录取通知了，然后发了所有人的考核成绩，一共200个人，我上机第9，理论面试第7，offer稳稳在手。但是因为答应了南大的offer，所以很快就给中科大的招生老师发消息拒绝了offer。

至此，整个7月的夏令营经历就是这些。

到了九月，开始了预报名，也就是九推之前的最后一波招生。但是（划重点），只有浙大没有招过生，其他学校都在夏令营招了一波。这意味着夏令营里已经基本录满，即使别人鸽了，还有waiting list里有人，况且计算机如此火热，基本不会在九推招生了。所以我就只报名了浙大的。后来整个华科计算机学院只有1个人过了浙大预报名审核，其余都没有过，当时一个年级14的同学报了也没有过。甚至前10也有人报没过(不确定)。整个华科只有5个人过了，两个电信的，两个软院的，1个计院的。我就好奇为啥，后来看到官网公布的入营名单，发现70%都是本校的学生。去年我有一个学长去了浙大，去年华科计院入营也只有3个，不是很多，后来只录取了1个人。我当时还想着浙大保底呢，后来还是经历了社会的挨打才觉得去浙大算是非常不错的了。

#### 最后一些看法

- 如果你要保内，尽量在暑假就要联系实验室和好的老师，否则等保研结果出来再联系就有点迟了
- 如果你非常想保外，报中科院录取的概率更大一些，因为他们没有本科生。像上交、复旦、浙大、南大他们本科生自己都已经占了70%的人，外校想进来还是有点难。
- 如果实在想换个地方读书，且排名30以后，可以试一下同等级的学校，比如哈工大、哈工深、上科大的优质导师。
- 夏令营最好就拿到offer，不然等九月份还没offer就非常危险。
- 每年填系统的时候，如果你放弃保研本校，可以冒险冲一下，在系统里填其他学校，如果有人鸽并且waiting list也没了，那么他们会在系统中面试志愿是他们学校的学生。但是这意味着如果你的志愿学校没有录取你，华科也一定不会录取你的，你就没学上了，白浪费一个保研资格，而且这样的风险极大，出现的概率还挺高，强烈不建议这样做。
